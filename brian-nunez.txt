    1. What is Semantic HTML?
    Semantic HTML is code that is simply to understand when marking up the website. For example, header means the head of the website, Footer means the bottom of the website etc.
    2. What is HTML used for? 
     To mark up the bones of the website you are building.
    3. What is an attribute and where do we put it? 
    An attribute works like name/value pairs. It can assign values to an element.
    4. What is the h1 tag used for? How many times should I use it on a page?
    h1 tag is used to indicate the name of the company or the name of the organization that the website is demonstrating. It should only be used once in the page. For example, you wouldn't have two titles in a book on the front cover
